什么是 Bootstrap

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目

Bootstrap 有哪些特点

  • 让前端开发更快速、简单
  • 所有开发者都能快速上手
  • 所有设备都可以适配
  • 所有项目都适用
